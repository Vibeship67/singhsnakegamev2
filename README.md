# Funny Snake Game

A classic snake game implemented in Python using Pygame. When you lose, it makes fun of you!

## Installation

1. Install Python if not already installed.
2. Install dependencies: `pip install -r requirements.txt`

## How to Run

Run the game with: `python main.py`

## Controls

- Arrow keys to move the snake
- Eat the red food to grow
- Avoid hitting walls or yourself

## Game Over

If you lose, the game will display a mocking message. Press C to play again or Q to quit.
